Proyecto Activo

Descripción del Proyecto

Este proyecto integra servicios de AWS para crear un sistema completo de análisis de datos. Implementa arquitectura serverless con Lambda, almacenamiento en S3, bases de datos RDS, y visualización interactiva mediante dashboards desarrollados en Python.

Stack Tecnológico

AWS Lambda Amazon S3 Amazon RDS Python Pandas Streamlit Jupyter Notebook AWS Secrets Manager

Arquitectura del Sistema

El diagrama muestra la arquitectura completa del sistema, incluyendo el flujo de datos desde la ingesta hasta la visualización, pasando por el procesamiento serverless y el almacenamiento en la nube.

Componentes del Proyecto

Funciones Lambda

Funciones serverless para procesamiento de datos

KP-Lambda.py

KP-Lambda encargarda de agrupar y transformar los datos (KPI) listos para el Dashboard

MC-Lambda.py

MC-Lambda, encargada de limpiar y normalizar los datos de origen a datos limpios en formato Parquet

Dashboard

Visualización interactiva de datos del Metro CDMX

Dashboard_metro.py

Dashboard de análisis del Metro CDMX
